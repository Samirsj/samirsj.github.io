Projet Application




Cette application est une solution simplifiée pour la gestion du personnel d'une organisation, développée en ligne de commande selon une architecture 3-tiers et conçue pour évoluer vers une application multi-utilisateurs grâce à une base de données. Elle repose sur trois niveaux d'habilitation distincts :


  • Employé standard : accès en lecture uniquement.

  • Administrateur de ligue : gestion des employés de sa ligue.

  • Super-administrateur (ROOT) : contrôle global.
